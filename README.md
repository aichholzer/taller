# Taller: API

* Introducción: APIs de alto rendimiento.
* Diseño de protocolos de seguridad.
* Manejo de sesiones.

### Requisitos

* Nodejs (7+) & NPM
* MongoDB
* Redis

```
> mkdir taller
> cd !$ && git clone git@github.com:aichholzer/taller.git .
> npm install && npm install -g grunt-cli
```

```
> cp .env.sample .env
```

```
> grunt nodemon
```
